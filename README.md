# RSDG

Random Starbucks Drink Generator

# Product Vision
For coffee advocates and Starbucks regulars who need a different beverage from their regular or go to, The RSDG is a drink creator and recommendation application that creates personalized recommendations based off past drinks and preferences. Unlike other drink creators, ours gives an individualized experience which caters to the users tastes.
